# Clock App

A clock app with random quotes, location info, & dynamic background based on the time of the day.

![](./screenshot_morning.png) ![](./screenshot_afternoon.png) ![](./screenshot_evening.png) ![](./screenshot_night.png)

## Tech Stack
- React
- Next.js
- Tailwind
